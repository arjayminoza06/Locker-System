 ## 📌 System Approved: Locker Management System

## 📌 System Description
This system is designed to manage locker assignments efficiently. It allows administrators to assign, monitor, and manage lockers for students. The system helps track available, occupied, and maintenance lockers while improving organization and reducing manual work.

## 📌 System Flow (Overview)

1. Admin logs into the system  
2. Admin searches student using Student ID  
3. System displays student details  
4. Admin assigns locker to student  
5. System updates locker status:
   - Available → Occupied  
6. System records the assignment  
7. Admin can monitor:
   - Active assignments  
   - Available lockers  
   - Occupied lockers  
   - Maintenance lockers  
8. Admin can also view full assignment history
